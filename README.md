# Man-Go JSON Web Framework

A lightweight, high-performance web framework building in **Go** designed for JSON-first applications. This project demonstrates modern backend patterns, including RESTful routing, ORM integration with PostgreSQL, and concurrent request handling.

Creating A new Module / APP  :: projectroot/myapp

 Create a Folder myapp in project root :: projectroot/myapp.json

 Create a Module / APP file myapp.json in project root
 
 Update the myapp.json file and run
 Example:

  <pre> [
    {
     "class":"content",
     "content":"Hello World"
    },
    {
     "class":"subtilte",
     "content":"Buliding Application in GO"
    }
   ]
   </pre>

 create a  header_scripts.txt, footer_scripts.txt files in myapp folder :: projectroot/myapp/header_scripts.txt, projectroot/myapp/footer_scripts.txt

 calling app http://localhost:8000/?proj=myapp
